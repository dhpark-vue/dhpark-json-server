![image](https://github.com/dhpark-vue/dhpark-json-server/assets/76561901/b59b650c-ca7a-44b3-b0f0-8a2dc908dbc3)



```
// install
npm install json-server

// create json
vim db.json

// run
npx json-server db.json

// deploy
fly deploy
```
